a MQTT 3.1.1 codec implemented by Java & Netty

R.I.P Dr.Li WenLiang

- Fast
- Easy of use
- Fully compliant to MQTT 3.1.1
- MQTT 5 support is on the way

# Packet Supported
- [x] CONNECT
- [x] CONACK
- [x] PUBLISH
- [x] PUBACK
- [ ] PUBREC
- [ ] PUBREL
- [ ] PUBCOMP
- [x] SUBSCRIBE
- [x] SUBACK
- [x] UNSUBSCRIBE
- [x] UNSUBACK
- [x] PINGREQ
- [x] PINGRESP
- [x] DISCONNECT
